# 📊 Customer Churn Analysis using SQL & Machine Learning

## 📌 Project Overview

Customer churn is a critical issue in the telecom industry, where customers stop using services.
This project analyzes customer data using **SQL** and builds a **Machine Learning model** in Python to predict churn and provide actionable business insights.

---

## 🎯 Objectives

* Analyze customer data using SQL queries
* Identify patterns and factors affecting churn
* Build a predictive model using Machine Learning
* Segment customers based on behavior
* Provide recommendations to reduce churn

---

## 🗂️ Dataset

Dataset used: **Telco Customer Churn Dataset**

### Features:

* CustomerID
* Age
* Gender
* Tenure
* MonthlyCharges
* Contract
* PaymentMethod
* TotalCharges
* Churn (Target Variable)

---

## 🧮 SQL Analysis

Key queries performed:

* Total number of customers
* Churn vs Non-Churn count
* Average Monthly Charges
* Tenure analysis
* Payment method analysis

SQL was used to extract meaningful insights from raw data.

---

## 🤖 Machine Learning Model

Model Used: **Random Forest Classifier**

### Why Random Forest?

* High accuracy
* Handles large datasets
* Reduces overfitting

---

## 📈 Results

* Model successfully predicts customer churn
* Important features identified:

  * MonthlyCharges
  * Tenure
  * Contract Type

---

## 👥 Customer Segmentation

Customers were classified into:

* **At Risk** → Likely to churn
* **Loyal** → Long-term customers
* **Dormant** → Low engagement

---

## 💡 Key Insights

* Customers with high monthly charges are more likely to churn
* New customers (low tenure) churn more
* Contract type impacts customer retention

---

## 🧾 Recommendations

* Provide offers to high-risk customers
* Encourage long-term contracts
* Improve customer service quality
* Optimize pricing strategies
* Use predictive model for early churn detection
